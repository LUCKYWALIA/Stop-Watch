# Stopwatch Web App #

A simple and interactive **Stopwatch Web Application** built using  
**HTML, JavaScript, and Tailwind CSS**.

This project demonstrates core JavaScript concepts like:
- setInterval()
- DOM manipulation
- Event handling
  
---

## Features

- Start / Stop stopwatch
- Lap time recording
- Reset functionality
- Millisecond precision
- Clean and responsive UI using Tailwind CSS

---

## Technologies Used

- **HTML5**
- **JavaScript (ES6)**
- **Tailwind CSS (via CDN)**
